CORE FEATURES

三件事,把工地的數字管起來

01 · 標單就是脊椎

PCCES 標單直接匯入,數字不再重打

把機關發的 eTender 預算書 XML 拖進來,幾千筆工項自動長成階層樹。 之後的日誌、估驗、進度、成本全部掛在同一套工項上——數量永遠對得起來。

  • 發包/非發包自動判別,金額權重自動計算
  • 變更設計(追加減帳)核准後,估驗與進度分母自動改用變更後契約金額
  • AI 解析契約,自動列出時程義務與罰則,到期前主動提醒
02 · 拍白板,日誌自動填

現場拍一張白板,AI 幫你寫日誌

工地白板寫好工項和數量、跟現場一起拍,AI 自動辨識並對回標單工項, 當日完成數量直接入帳。工程師少打字,資料反而更完整。

  • 一天一筆,附現場照片,手機瀏覽器直接用
  • 日誌數量自動累計,月底估驗「一鍵帶入」
  • 施工月報自動彙整當月進度、品質、工安
03 · 估驗到請款,錢的事不漏接

估驗計價自動算,S 曲線即時看

本期金額、累計完成度、保留款、應付金額自動計算,估驗單直接列印/存 PDF。 S 曲線比對預定與實際進度,落後工項自動排序,請款收款進度一頁掌握。

  • 估驗流程:草稿 → 送審 → 監造審核 → 已核定 → 已請款
  • 成本毛利即時對照,分包發包一併管理
  • 逾期未請款、已請款未收款,提醒中心自動盯
